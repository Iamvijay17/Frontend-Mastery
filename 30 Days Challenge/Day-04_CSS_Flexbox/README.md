# Day 04 - CSS Flexbox

## Goal
Master one-dimensional layouts using Flexbox.

## Topics to Learn
- display: flex
- flex-direction: row and column
- justify-content: flex-start, center, space-between, space-around, space-evenly
- align-items: stretch, center, flex-start, flex-end
- flex-wrap and gap
- flex-grow, flex-shrink, flex-basis
- align-self on individual items
- order property

## Practice Task
Build flexbox.html:
- A horizontal navbar with logo on the left and links on the right
- A row of 4 cards that wraps on smaller screens
- A centered hero section aligned both horizontally and vertically

## Resources
- https://css-tricks.com/snippets/css/a-guide-to-flexbox/
- https://flexboxfroggy.com (game to practice!)

## Pro Tip
Flexbox is 1D - it works along one axis at a time. For 2D page layouts, use CSS Grid.
