# Day 01 - HTML Fundamentals

## Goal
Build a strong foundation in HTML structure and core tags.

## Topics to Learn
- What is HTML? How browsers render pages
- DOCTYPE, html, head, body structure
- Headings h1-h6, paragraphs, line breaks
- Bold, italic, mark, small, del tags
- Links with anchor tags and Images with img
- HTML Comments

## Practice Task
Create index.html - a personal intro page with:
- Your name as h1 heading
- A short bio paragraph
- A profile image using a placeholder URL
- Links to 3 of your favourite websites

## Resources
- https://developer.mozilla.org/en-US/docs/Web/HTML
- https://www.w3schools.com/html/

## Pro Tip
Every HTML page needs DOCTYPE html at the top. It tells the browser to use HTML5 standards.

## Checklist
- [ ] Understand HTML document structure
- [ ] Use at least 8 different tags
- [ ] Validate your page at https://validator.w3.org
