# Day 12 - JavaScript: ES6+ Modern Syntax

## Goal
Write clean, modern JavaScript that is used in every professional codebase today.

## Topics to Learn
- Arrow functions and implicit return for one-liners
- Template literals for string interpolation
- Array and object destructuring
- Spread (...) and rest (...) operators
- Default function parameters
- Optional chaining with obj?.property?.nested
- Nullish coalescing: value ?? fallback
- Short-circuit evaluation: value && doSomething()
- ES Modules: import and export (default and named exports)
- for...of loops for iterables
- Object.keys(), Object.values(), Object.entries()

## Practice Task
Go back to your JS files from Days 8-11 and refactor them:
- Replace all function() declarations with arrow functions
- Apply destructuring wherever you access object properties
- Replace all string concatenation with template literals
- Add optional chaining to any nested property access from APIs

## Pro Tip
Optional chaining ?. prevents the infamous Cannot read properties of undefined error. Use it whenever accessing data from an API response.
