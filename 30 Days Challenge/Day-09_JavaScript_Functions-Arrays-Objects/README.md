# Day 09 - JavaScript: Functions, Arrays and Objects

## Goal
Write reusable code with functions and manage collections of data with arrays and objects.

## Topics to Learn
- Function declarations vs expressions vs arrow functions
- Parameters, arguments, default values, and rest parameters
- Return values
- Array creation, access by index, push, pop, shift, unshift, splice
- Array methods: map, filter, reduce, find, findIndex, includes, some, every, flat
- Spread operator with arrays: [...arr]
- Objects: key-value pairs, dot notation, bracket notation
- Object destructuring: const { name, age } = person
- Spread with objects: { ...obj, newKey: value }
- Working with arrays of objects

## Practice Task
Create data.js:
- A function that filters an array of products by a given price range
- A function using reduce to calculate the total price of a shopping cart
- An array of 5 user objects - use map to extract only the names

## Pro Tip
map, filter, and reduce are the three most important array methods in JavaScript. Master these before moving on.

## Checklist
- [ ] Write at least 5 arrow functions
- [ ] Use map, filter, and reduce on real data
- [ ] Destructure both objects and arrays
