# Day 06 - Responsive Design and Media Queries

## Goal
Make every page look great on mobile, tablet, and desktop screens.

## Topics to Learn
- Viewport meta tag
- Mobile-first design approach
- Media queries with min-width and max-width
- Responsive units: %, vw, vh, rem, em, clamp()
- Responsive images with max-width: 100% and object-fit
- Common breakpoints: 480px, 768px, 1024px, 1280px
- Hiding and showing elements at different breakpoints

## Practice Task
Take your Day-05 grid layout and make it fully responsive:
- Mobile: single column stacked layout
- Tablet: 2 column layout
- Desktop: full 3-column layout with sidebar

## Pro Tip
Always start mobile-first. Write base styles for small screens, then use min-width media queries to scale up to larger screens.

## Checklist
- [ ] Add the viewport meta tag to HTML
- [ ] Write 3 breakpoints for mobile, tablet, and desktop
- [ ] Test your layout in Chrome DevTools device emulator
