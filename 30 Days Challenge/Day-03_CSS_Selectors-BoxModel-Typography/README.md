# Day 03 - CSS Selectors, Box Model and Typography

## Goal
Style HTML pages with CSS. Understand how elements take up space and how to style text beautifully.

## Topics to Learn
- Inline, internal, and external CSS
- Selectors: element, class, ID, grouping, descendant, child
- Specificity rules
- Box model: content, padding, border, margin
- box-sizing: border-box
- display: block, inline, inline-block
- font-family, font-size, font-weight, line-height, letter-spacing
- Google Fonts integration
- Colors: hex, rgb, hsl, rgba

## Practice Task
Create style-demo.html and style.css:
- Style a blog article with a Google Font
- Create a card with padding, border, margin, and box shadow
- Use at least 5 different CSS selectors

## Pro Tip
Add * { box-sizing: border-box; margin: 0; padding: 0; } at the top of every CSS file. It saves hours of layout pain.

## Checklist
- [ ] Link an external CSS file correctly
- [ ] Understand the difference between margin and padding
- [ ] Use a Google Font on the page
