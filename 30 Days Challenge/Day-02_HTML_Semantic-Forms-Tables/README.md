# Day 02 - HTML Semantic Elements, Forms and Tables

## Goal
Write meaningful HTML using semantic tags, build forms, and display tabular data.

## Topics to Learn
- Semantic tags: header, nav, main, section, article, aside, footer
- Lists: ul, ol, li, nested lists
- Tables: table, thead, tbody, tr, th, td, colspan, rowspan
- Form inputs: text, email, password, number, checkbox, radio, date
- select, textarea, button, label, placeholder, required attributes

## Practice Task
Create portfolio.html with:
- Semantic layout using header, nav, main, and footer
- A skills table showing skill name, level, and years of experience
- A contact form with name, email, message fields and a submit button

## Pro Tip
Always pair label for="id" with input id="id". This improves accessibility and UX massively.

## Checklist
- [ ] Use at least 6 semantic tags
- [ ] Build a table with 3 or more columns
- [ ] Build a form with 4 or more input types
