# Day 13 - MINI PROJECT: Responsive Landing Page

## Goal
Apply everything from Days 1-12 to build one polished, real-world project.

## What to Build
A fully responsive personal or product landing page.

## Required Sections
- Navbar with logo and navigation links, with a hamburger menu on mobile
- Hero section with a headline, subtitle, and call-to-action button
- Features or About section using a CSS Grid card layout
- A contact form with client-side validation
- Footer with links and social icons

## Technical Requirements
- Semantic HTML5 structure throughout
- CSS custom properties for the entire color theme
- Fully responsive layout for mobile, tablet, and desktop
- Smooth scrolling and hover transitions
- JavaScript form validation before submission
- Working mobile hamburger menu toggle

## Stretch Goals
- Dark mode toggle using CSS variables switched by JavaScript
- Scroll-triggered fade-in animations
- Deploy the finished project on GitHub Pages or Netlify

## Pro Tip
This is your first real portfolio piece. Spend extra time on the design and make it something you are proud to show employers.

## Checklist
- [ ] Fully responsive on all screen sizes
- [ ] All sections built and filled with content
- [ ] Form validates correctly before submission
- [ ] Hamburger menu works on mobile screens
- [ ] Project deployed and accessible via a public URL
