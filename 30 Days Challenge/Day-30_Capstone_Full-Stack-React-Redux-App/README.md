# Day 30 - CAPSTONE PROJECT: Full React and Redux Application

## Goal
Build a complete, deployable frontend application that demonstrates every skill from the 30-day challenge.

## Choose One of These Project Ideas
1. E-Commerce Store with product listing, a Redux-powered cart, and a checkout form
2. Job Board with job listings, filters by category, a save favourites feature, and an application form
3. Movie Database using the TMDB API with search, a personal watchlist, and ratings
4. Personal Finance Tracker with income and expense entries, category breakdowns, and summary charts

## Required Technology Stack
- React with Vite as the build tool
- TypeScript for all component and hook files
- Redux Toolkit for all global client state
- RTK Query for all server state and API data fetching
- React Router with at least 4 navigable pages
- Tailwind CSS for all styling
- shadcn/ui for common UI components
- At least 2 custom hooks extracted from component logic
- Fully responsive mobile-first layout

## Required Features
- At least 4 separate navigable pages
- Full CRUD operations for at least one type of data
- Loading state, error state, and empty state handled for all async data
- Client-side form validation with user-friendly error messages
- Consistent and polished UI across all pages

## Stretch Goals
- Mock authentication flow with login, logout, and protected routes
- Dark mode toggle that persists to localStorage
- Skeleton loading screens instead of spinners
- Unit tests written with Vitest and React Testing Library
- Deployed to Vercel or Netlify with a shareable URL

## Suggested Day Plan
Morning: Write out the feature list, set up the Vite project, install all packages, and configure the store and router.
Afternoon: Build the core layout, all route pages, and the Redux slices and RTK Query endpoints.
Evening: Implement the main features, polish the UI for all screen sizes, and deploy the finished app.

## Pro Tip
Do not aim for perfection before shipping. A working application that is deployed and accessible on the internet is worth infinitely more than a perfect application sitting only on your laptop.

## Final Checklist
- [ ] Project is planned and fully scaffolded
- [ ] All routes are working and navigable
- [ ] Redux store is set up with at least 2 feature slices
- [ ] At least one RTK Query API integration is complete
- [ ] Layout is fully responsive on mobile and desktop
- [ ] App is deployed and accessible via a public URL
- [ ] Project is pushed to GitHub and linked from your portfolio
