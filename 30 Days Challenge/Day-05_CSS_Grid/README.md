# Day 05 - CSS Grid

## Goal
Build two-dimensional layouts using CSS Grid.

## Topics to Learn
- display: grid
- grid-template-columns and grid-template-rows
- fr unit and repeat()
- gap, column-gap, row-gap
- grid-column and grid-row span
- minmax() and auto-fill and auto-fit
- grid-template-areas
- Placing items explicitly on the grid

## Practice Task
Build grid-layout.html:
- A full page layout with header, sidebar, main content, and footer using grid-template-areas
- A responsive image gallery using auto-fill and minmax

## Resources
- https://css-tricks.com/snippets/css/complete-guide-grid/
- https://cssgridgarden.com (game to practice!)

## Pro Tip
grid-template-columns: repeat(auto-fill, minmax(250px, 1fr)) creates a fully responsive grid with zero media queries.
